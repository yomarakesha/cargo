Процесс

Как это работает

От поиска товара на Taobao до доставки к вашей двери в Ашхабаде — 4 простых шага.

1
🔗

Вставьте ссылку

Скопируйте ссылку товара с Taobao, 1688 или Dewu — система автоматически парсит название, фото и цену.

2
💳

Оплатите в манатах

Видите итоговую сумму в TMT: цена товара + комиссия + доставка по Китаю. Оплата списывается с кошелька.

3
📦

Выкуп и контроль

Наш склад в Гуанчжоу принимает товар, фотографирует, взвешивает и отправляет вам уведомление с фото.

4
🏠

Доставка на дом

GPS-трекинг на всём маршруте. Курьер доставит заказ по вашему адресу в Ашхабаде с подтверждением.

Платформа

Полная экосистема

4 приложения в единой инфраструктуре — клиент, склад, курьер и администратор.

📱

Клиентское приложение

iOS + Android (React Native). Интерфейс на русском, туркменском и английском.

  • Выкуп с Taobao / 1688 / Dewu по ссылке
  • Авто-парсинг товара через ElimAPI
  • Кошелёк в манатах (TMT)
  • GPS-трекинг на карте в реальном времени
  • Push + SMS уведомления
  • Страхование товара при заказе
🏭

Складское ПО (WMS)

Для операторов склада в Китае и Туркменистане. Android планшет + Web.

  • Сканирование трек-номеров и QR-кодов
  • Фотофиксация при приёмке (мин. 2 фото)
  • Взвешивание и объёмный вес
  • Консолидация в мешки (Master-ID)
  • Привязка GPS-трекера к рейсу
  • Офлайн-режим с синхронизацией
📊

Административная панель

Полный контроль над платформой. React Web, ролевой доступ (RBAC).

  • Дашборд: заказы, выручка, рейсы в реальном времени
  • Управление курсом TMT/CNY ежедневно
  • Карта активных GPS-трекеров
  • Управление сотрудниками и ролями
  • Финансовые отчёты (Excel / PDF)
  • 2FA для администраторов
🛵

Приложение курьера

Для курьеров в Ашхабаде. iOS + Android (React Native).

  • Список доставок на сегодня с адресами
  • Навигация через Yandex Maps / Google Maps
  • Смена статуса: принял / в пути / доставлено
  • Фото у двери + электронная подпись клиента
  • История и статистика доставок

Маршруты

3 способа доставки

Выбирайте между скоростью и ценой — от бюджетной авто-доставки до авиа-экспресса.

🚛 Основной

Автодоставка

Гуанчжоу → Урумчи → Хоргос → Казахстан → Узбекистан → Ашхабад

⏱ Срок:  14–20 дней
📏 Расстояние:  5 000–7 000 км
📦 Тип груза:  LTL/FTL контейнеры
🏁 Граница:  Порт Хоргос (24/7)
💰 Стоимость:  Базовый тариф
✈️ Экспресс

Авиадоставка

Гуанчжоу / Пекин → Прямой рейс → Ашхабад

⏱ Срок:  2–5 дней
📦 Тип груза:  Мелкие, высокоценные
🎯 Применение:  Электроника, люкс
💰 Стоимость:  + наценка к базе
🚂 Альтернатива

Железнодорожная

Урумчи → Хоргос → Казахстан → Туркменистан

⏱ Срок:  15–20 дней
📦 Тип груза:  Крупные партии
🔮 Статус:  При масштабировании

Реализация

Дорожная карта

7 месяцев при параллельном выполнении этапов. Полный запуск — конец 2026 года.

Этап 0

Подготовка

1–2 недели
  • Найм юридической фирмы в Гуанчжоу
  • Поиск склада в Гуанчжоу
  • Подбор юриста (торговая лицензия)
Этап 1

Юридика + Склад

2 месяца
  • Регистрация WFOE
  • Аренда + оборудование склада
  • Alipay Business + WeChat Pay
  • GPS-трекеры Eelink GPT50
  • Договор с ElimAPI
Этап 2

Ядро системы

4 месяца
  • Архитектура БД и API
  • Shared Core (заказы, кошельки)
  • WMS (приёмка, консолидация)
  • GPS Tracking Service
  • Admin Panel
Этап 3–4

Клиент + Запуск

3 месяца
  • Клиентское приложение (React Native)
  • Courier App
  • Altyn Asyr онлайн-платежи
  • QA + Нагрузочное тестирование
  • Пилот: 10–20 реальных заказов
